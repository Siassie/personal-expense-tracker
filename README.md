This is a personal Expense Tracker application built with Python and FastAPI. 
It allows users to securely sign up, log in, add expenses, and view their expenses by type or individually. 
The project uses JWT for authentication, SQLAlchemy ORM with PostgreSQL for data storage, and provides a clean Command-Line Interface (CLI) for interaction. 
Designed with modular architecture, it demonstrates modern Python backend development, secure authentication practices, and API-driven development. 
Ideal for learning full-stack backend development and building RESTful APIs with proper database integration.
